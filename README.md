# chamuka-skills

A Claude Code plugin of reusable agent skills — techniques, patterns, and reference guides.

## Install

```
claude plugin add chamuka-inc/chamuka-skills
```

## Skills

Skills live in `skills/<skill-name>/`. Each skill contains a `SKILL.md` with YAML frontmatter and markdown instructions.

No skills yet — this repo is freshly scaffolded.
